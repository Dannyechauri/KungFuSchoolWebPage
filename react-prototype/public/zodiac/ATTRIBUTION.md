# Chinese zodiac artwork

The zodiac carousel uses artwork from the Wikimedia Commons collection
[`Chinese zodiac icons`](https://commons.wikimedia.org/wiki/Category:Chinese_zodiac_icons).

- [`Rat`](https://commons.wikimedia.org/wiki/File:Rat.svg),
  [`Ox`](https://commons.wikimedia.org/wiki/File:Ox.svg),
  [`Tiger`](https://commons.wikimedia.org/wiki/File:Tiger.svg),
  [`Rabbit`](https://commons.wikimedia.org/wiki/File:Rabbit.svg),
  [`Snake`](https://commons.wikimedia.org/wiki/File:Snake.svg),
  [`Horse`](https://commons.wikimedia.org/wiki/File:Horse.svg),
  [`Goat`](https://commons.wikimedia.org/wiki/File:Goat.svg) and
  [`Monkey`](https://commons.wikimedia.org/wiki/File:Monkey.svg): public domain.
- [`Dragon`](https://commons.wikimedia.org/wiki/File:Dragon.svg),
  [`Rooster`](https://commons.wikimedia.org/wiki/File:Rooster.svg),
  [`Dog`](https://commons.wikimedia.org/wiki/File:Dog_2.svg) and
  [`Boar`](https://commons.wikimedia.org/wiki/File:Boar.svg): created by Wikimedia
  Commons user [ANGELUS](https://commons.wikimedia.org/wiki/User:ANGELUS) and licensed under
  [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/).

The original artwork has not been modified. The interface applies color and
opacity filters at display time.

The cycle separator uses
[`Pakua.svg`](https://commons.wikimedia.org/wiki/File:Pakua.svg), attributed to
[BenduKiwi](https://commons.wikimedia.org/wiki/User:BenduKiwi) and licensed
under [CC BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/). Its
original black, white and grey palette was adapted to red and blue for this
interface.
